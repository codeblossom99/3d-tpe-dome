# 3D 台北大巨蛋

[English](./README.md)

![3D 台北大巨蛋場館預覽](./tpe-demo.png)

這是一個以台北大巨蛋為概念製作的互動式 3D 場館地圖，讓使用者能更直覺地了解座位區域的位置，並在抵達場館前先熟悉周遭配置。

## 功能特色

- 透過互動式 3D 視角探索場館
- 搜尋並選擇座位區域
- 使用小地圖快速了解各區位置
- 從所選區域切換至第一人稱視角
- 擷取並下載目前畫面的截圖

## 開始使用

安裝相依套件：

```bash
npm install
```

啟動開發伺服器：

```bash
npm run dev
```

接著在瀏覽器開啟 [http://localhost:3000](http://localhost:3000)。

## 技術棧

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Three.js](https://threejs.org/)
